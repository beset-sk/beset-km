SpringBoot compatibility version matrix ( https://endoflife.date/spring-boot )

For java 11 we choose SpringBoot ver. 2.7.18 (the last one compatible with java 11) 3.x need java 17+
Gwt Web Toolkit for SpringBoot - https://github.com/levigo/gwt-spring-boot-starter
    * follow Sample https://github.com/levigo/gwt-spring-boot-starter/tree/master/example


Hibernate Migration - SpringBoot is referenced Hibernate 5 via package SpringDataJPA (https://mvnrepository.com/artifact/org.springframework.data/spring-data-jpa/2.7.18)
So supported version for org.hibernate is 5.6.14.Final+ . We have to realized a migration from 3.x to 5.x hibernate
    * follow next tutorial https://medium.com/@dheeraj_kumbar/hibernate-3-x-to-5-x-73f27871309b
